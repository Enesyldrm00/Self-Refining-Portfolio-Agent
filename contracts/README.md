# Self-Refining Portfolio Agent - Contracts

This folder contains the Soroban Rust smart contracts for the Self-Refining Portfolio Agent.

## Structure

```
contracts/
├── portfolio-manager/    # Main portfolio management contract
├── strategy-optimizer/   # AI strategy optimization contract
└── swap-executor/        # DEX swap execution contract
```

## Setup

Prerequisites:
- Rust toolchain
- Soroban CLI
- Stellar testnet access
